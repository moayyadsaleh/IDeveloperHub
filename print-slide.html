One-Click Slide Printing for E-Learning

Ever wish your learners could just hit “print” and walk away with exactly what they need? Now they can! Whether it’s printing slides for note-taking, grabbing key takeaways, or saving certificates, this one-click print feature makes it effortless. No more screenshots, no more copy-pasting. It works perfectly with Articulate Storyline, Adobe Captivate, or Lectora – even if those tools don’t have a built-in print option. One click, and boom – your learners have what they need, on paper and ready to go!

DESCRIPTION

This print feature helps instructional designers give learners the option to easily print slides, notes, certificates, or key takeaways in eLearning courses. No more tedious screenshots or copying text! Just add the print button, and your learners can save or print what they need with a single click.

SEE IT WORKING IN A COURSE

Want to see the print feature in action? Check out this LIVE DEMO I created in an Articulate Storyline course. It shows how learners can print certificates, slides, or notes effortlessly. Go ahead, give it a try and see how it can make your courses more practical and user-friendly!

HOW TO USE

  1. Copy the Print Code: Here’s the JavaScript code you need to add a print function with automatic scaling:
    var style = document.createElement('style');
    style.innerHTML = `
      @media print {
        body {
          transform: scale(0.8);
          transform-origin: top left;
          width: 100%;
        }
    }
    `;
    document.head.appendChild(style);
    window.print();
  2. Go to Your Authoring Tool: Open your preferred authoring tool like Articulate Storyline, Adobe Captivate, or any other tool that supports JavaScript.
  3. Add a Button or Print Icon: - Insert a button labeled “Print This Page”.
    - For a modern look, use a sleek printer icon instead of text.
    - In Storyline: Go to Insert > Icons and search for a printer icon.
    - In Captivate: Insert an image of a printer icon.
  4. Add a Trigger to the Button: - In Articulate Storyline:
    1. Select the button or icon.
    2. Open the Triggers Panel.
    3. Click “Create a New Trigger”.
    4. Set the options:
    - Action: Execute JavaScript
    - When: User clicks the button
    - In Adobe Captivate:
    1. Select the button or icon.
    2. Go to the Properties Panel.
    3. Under the Actions tab, set On Success to “Execute JavaScript”.
  5. Paste the JavaScript Code: In the JavaScript editor, paste the code you copied earlier:
    var style = document.createElement('style');
    style.innerHTML = `
      @media print {
        body {
          transform: scale(0.8);
          transform-origin: top left;
          width: 100%;
        }
    }
    `;
    document.head.appendChild(style);
    window.print();
  6. Publish and Test: - Publish Your Project:
    - In Storyline: Go to Publish > Web or your preferred format.
    - In Captivate: Go to Publish > Publish to HTML5.

    - Test the Print Function:
    1. Open the published project in a web browser (Chrome, Firefox, or Edge).
    2. Click the print button or icon.
    3. The print dialog should pop up, and the content should scale to fit the page perfectly.
