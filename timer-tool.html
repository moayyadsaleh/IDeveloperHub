Modern Sleek Phone-like Timer Tool

Add a sleek countdown phone-like timer to your eLearning projects! Perfect for quizzes, timed instructor-led activities, or pacing learners in tools like Articulate Storyline, Adobe Captivate, or Lectora. This timer comes with a transparent background to blend with any theme color you're using.

DESCRIPTION

This timer tool helps instructional designers add customizable countdowns to eLearning courses. It’s great for keeping learners on track during timed activities or assessments. Just set the duration, embed it, and you're good to go!

SEE IT WORKING IN A COURSE

Want to see the timer in action? Check out this LIVE DEMO I created in an Articulate Storyline course. It gives you a quick feel for how the timer works and how sleek it looks in a real project. Go ahead, take it for a spin!

HOW TO USE:

  1. DOWNLOAD THE TIMER CODE FOLDER: Click the download button below to get the zipped folder containing the timer code.
  2. Unzip the Folder: - On Windows: Right-click the downloaded .zip file and select "Extract All…". - On macOS: Double-click the .zip file to automatically unzip it.
  3. Open Your Authoring Tool (Articulate Storyline): - Open your Articulate Storyline project. - Go to the Insert tab and select Web Object.
  4. Add the Timer Folder as a Web Object: - In the Web Object dialog, click Browse. - Select the unzipped timer folder (not an individual file, but the entire folder). - Click OK to insert the web object into your slide.
  5. Customize the Look: - Add a shape in your project to match your theme color. - Place the timer web object on top of the shape and position it however you like.
  6. Preview and Test: - Preview your project to ensure everything works smoothly.
