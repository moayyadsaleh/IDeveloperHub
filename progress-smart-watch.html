E-learning progress Tracker

I designed this interactive tool for my fellow instructional designers who want to make their e-learning courses more engaging. It‚Äôs a great fit for self-paced learning because it tracks progress automatically as learners complete each module or task. Inspired by the sleek look of smartwatches, it features fun visuals, sound effects, and confetti to keep learners excited and show them how far they‚Äôve come. Plus, progress is saved on their device, so they can easily pick up where they left off. It‚Äôs a simple, effective way to keep learners motivated and celebrate their milestones!

DESCRIPTION

The Progress Tracker is designed to perfectly integrate into any e-learning course. It features a solid white background to ensure it blends well with various designs. Once embedded, you can trigger progress updates in multiple ways: by adding a Finish Chapter button, a Complete Course button, after a quiz, or when the timeline ends on a slide. This flexibility lets you signal progress whenever learners hit a key milestone.

To ensure progress doesn‚Äôt increment more than once when the same button is clicked, you can add a trigger to disable the button after use.

HOW TO USE

  1. Download the Zip File:
    Click the download button below to get the Progress Tracker zip file. This zip contains all the necessary code and assets.
  2. Unzip the File:
    • On Windows: Right-click the downloaded .zip file and select "Extract All...".
    • On macOS: Double-click the .zip file to automatically unzip it.
    After unzipping, you'll get a folder named progress-tracker.
  3. Open Your E-Learning Project:
    Open your course in Articulate Storyline or any other authoring tool that supports embedding web objects.
  4. Insert the Web Object:
    • In Articulate Storyline:
      • Go to the Insert tab and select Web Object.
      • In the Web Object dialog, click Browse.
      • Select the unzipped progress-tracker folder (the entire folder, not individual files).
      • Click OK to embed the web object into your slide.
    Position the web object where you want the Progress Tracker to appear (e.g., at the end of a module or chapter).
  5. Add a Button to Signal Completion:
    Add a button (e.g., ‚ÄúFinish Chapter‚Äù or ‚ÄúComplete Module‚Äù) on the same slide where the web object is embedded. This button will trigger the progress update.
  6. Add JavaScript to Trigger Progress Updates:
    To ensure progress updates correctly and doesn‚Äôt increment multiple times, use the following JavaScript:

    JavaScript Code:

          
sessionStorage.removeItem("progressUpdated");
var webObject = document.querySelector('iframe');
if (webObject && !sessionStorage.getItem("progressUpdated")) {
  webObject.contentWindow.postMessage("chapterCompleted", "*");
  sessionStorage.setItem("progressUpdated", "true");
}

    How to Add This Trigger in Storyline:

    • Select your "Complete" button.
    • Go to the Triggers panel and create a new trigger to Execute JavaScript when the button is clicked.
    • Paste the code above into the JavaScript window.
    • Click OK to save the trigger.
  7. Change the Default Number of Sections:
    By default, the Progress Tracker is set to 5 sections. To change this, modify the following line in your JavaScript: 
          
const totalSections = 3; // Change this number to your desired total
    Replace 3 with the number of sections you want to track. For example, to track 5 sections, update it to: 
          
const totalSections = 5;
  8. Prevent Duplicate Progress Updates:
    To ensure progress doesn‚Äôt increment if the button is clicked again:
    • Add another trigger to the button:
      • Set state of the button to Disabled when the user clicks the button.
    This prevents the button from being clicked multiple times.
  9. Repeat for Additional Chapters or Modules:
    • For each new chapter or module, copy the web object embed and add a new button.
    • Use the same JavaScript and state triggers for each button to manage progress updates.
  10. Test Your Course:
    • Preview your course to ensure the Progress Tracker updates correctly when the button is clicked.
    • Verify that progress is saved and doesn‚Äôt increment more than once per click.
    • Check that the button becomes disabled after being clicked.
  11. Save Progress Automatically:
    The Progress Tracker uses sessionStorage to manage progress updates. This ensures progress is only updated once per session and resets if the session is closed.
  12. Celebrate Completion:
