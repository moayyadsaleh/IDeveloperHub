E-learning progress Tracker

I created this interactive tool for my fellow instructional designers to embed in e-learning courses. It’s perfect for self-paced learning, automatically tracking progress as learners complete each module or task. It's inspired by the look and feel of smartwatches, it uses fun visuals, sounds, and confetti to keep learners excited and show them how far they’ve come. Progress is saved on their device, so they can pick up where they left off. It’s a simple way to keep learners motivated and celebrate their achievements along the way.

DESCRIPTION

Progress Tracker is designed to fit perfectly into any e-learning course. It has transparent background to make it blend perfectly with your design. Once embedded, you can trigger it in multiple ways: with a Complete Chapter button, a Complete Course button, after a quiz, or even when the timeline ends on a slide. This flexibility means you can update progress whenever learners hit a key milestone.

The progress is automatically saved on your device using local storage, so if you’re testing the meter while building your course, don’t get confused if the progress seems to carry over. It’s working as intended! If you want to reset it for testing purposes, clear your browser’s local storage or open the course in an incognito window.

This progress tracker is perfect for self-paced courses because it visually tracks up to five milestones which keeps learners motivated. When learners complete all milestones, they’ll be rewarded with celebratory confetti and a cheerful yay-and-clap sound. To use it across different milestones, you can simply copy the same progress page or web object and paste it where needed. Each time you trigger progress, the meter updates dynamically to give learners a clear, engaging sense of accomplishment.

HOW TO USE

  1. Download the Zip File:
    Click the download button below to get the Progress Tracker zip file. This file contains all the necessary code and assets.
  2. Unzip the File:
    - On Windows: Right-click the downloaded .zip file and select "Extract All...".
    - On macOS: Double-click the .zip file to automatically unzip it.
    After unzipping, you'll get a folder named progress-tracker.
  3. Open Your E-Learning Authoring Tool:
    Open your e-learning project in tools like Articulate Storyline, Adobe Captivate, or any other tool that supports web objects.
  4. Insert a Web Object:
    - In Articulate Storyline:
    - Go to the Insert tab and select Web Object.
    - In the Web Object dialog, click Browse.
    - Select the unzipped Progress Tracker folder (select the entire folder, not individual files).
    - Click OK to insert the web object into your slide.
    - In Adobe Captivate:
    - Go to the Media menu and select Web Object.
    - Enter the path to the unzipped Progress Tracker folder or browse to it.
    - Click OK to insert the web object.
  5. Trigger Progress Updates with JavaScript:
    To update the Progress Tracker when learners complete a chapter, task, or quiz, add a trigger to Execute JavaScript with the following code:

    Copy and Paste This Trigger:

                  document.querySelector('iframe').contentWindow.postMessage('chapterCompleted', '*');

    When to Use This Trigger:

    • When the timeline ends on a slide.
    • When learners click a “Complete” button.
    • After a quiz or interaction is completed.
    This ensures the Progress Tracker updates automatically and saves progress whenever a milestone is reached.
  6. Position the Tracker:
    Move and resize the web object on your slide to fit your design. Place it at the end of modules, tasks, or wherever you want learners to see their progress.
  7. Progress is Automatically Saved:
    The Progress Tracker automatically saves progress on the learner’s device using local storage. This means:
    - Learners can close the course and pick up where they left off if they use the same device and browser.
    - If you’re testing the tracker while building your course, progress will persist. To reset progress during testing, clear your browser’s local storage or use an incognito window.
  8. Celebrate Completion:
    Once all milestones are complete, the tracker rewards learners with:
    - Confetti animation 🎉
    - A cheering sound effect
    - A congratulatory message: “Successfully completed, congrats!”
  9. Track Up to 5 Milestones by Default:
    The Progress Tracker is set to track 5 milestones by default. If you need to track more, you can modify the code:
    1. Open the HTML file in a text editor like Notepad or Visual Studio Code.
    2. Find this line in the script section:
    ```javascript const totalSections = 5; ```
    3. Change 5 to the number of milestones you need (e.g., 8).
    4. Save the file and test your course to ensure the updates work correctly.
  10. Preview and Test:
    - Preview your course to ensure the Progress Tracker works as expected.
    - Verify that progress updates, visuals, confetti, and sound effects trigger correctly.
    - Remember: Progress is saved locally, so learners will see their progress as long as they use the same device and browser.
