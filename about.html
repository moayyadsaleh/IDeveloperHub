ABOUT IDEVELOPERHUB

IDeveloperHub is a free platform, and it always will be. I’ve put together custom code snippets to simplify tricky tasks, make your eLearning projects more engaging, and add features that authoring tools just don’t offer. These tools are here to save you time, so you can focus on what really counts: creating learning experiences that stick.

I’m always adding new code snippets and resources to keep things fresh. I want my fellow instructional designers to have tools that don’t just make work easier; they make it more enjoyable, too.
