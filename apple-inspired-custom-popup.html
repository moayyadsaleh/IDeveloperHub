One-Click Pop-up

I created this tool to help my fellow instructional designers add sleek, custom pop-ups to e-learning courses in just one click. Need a quick message, a reminder, or some engaging feedback? No coding headaches or complicated setups, just click, and you’ve got a beautiful, ready-to-go pop-up. It works smoothly with Articulate Storyline, Adobe Captivate, and Lectora.

DESCRIPTION

Building modern pop-up dialogs in e-learning can be a real challenge. It often involves multiple layers and complex setups. That’s why I created this custom pop-up feature to make life easier for us. This tool blurs the background and smoothly adds a sleek, customizable pop-up to your course. It's inspired by Apple’s minimal design and adds a familiar, polished touch that learners will recognize and appreciate. No coding headaches, just click, add your content, and engage your learners effortlessly.

SEE IT WORKING IN A COURSE

Curious to see the pop-up feature in action? Check out this LIVE DEMO I created in an Articulate Storyline course. Go ahead, give it a try and see how your courses can pop (pun intended)!

HOW TO USE

  1. Copy the Pop-Up Code: Here’s the JavaScript code you need to add a sleek pop-up to your eLearning course:
    // Create a background overlay with blur effect
    const overlay = document.createElement('div');
    overlay.style.position = 'fixed';
    overlay.style.top = '0';
    overlay.style.left = '0';
    overlay.style.width = '100%';
    overlay.style.height = '100%';
    overlay.style.backgroundColor = 'rgba(0, 0, 0, 0.25)';
    overlay.style.backdropFilter = 'blur(10px)';
    overlay.style.zIndex = '9998';
    overlay.style.display = 'flex';
    overlay.style.justifyContent = 'center';
    overlay.style.alignItems = 'center';
    overlay.style.margin = '0';
    overlay.style.padding = '0';

    const popup = document.createElement('div');
    popup.style.backgroundColor = '#f8f8f8';
    popup.style.padding = '28px';
    popup.style.borderRadius = '20px';
    popup.style.boxShadow = '0 12px 30px rgba(0, 0, 0, 0.15)';
    popup.style.width = '400px';
    popup.style.maxWidth = '90%';
    popup.style.textAlign = 'center';
    popup.style.fontFamily = 'Helvetica, Arial, sans-serif';

    const heading = document.createElement('h2');
    heading.textContent = 'Welcome!';
    popup.appendChild(heading);

    const message = document.createElement('p');
    message.textContent = 'Experience a clean, sleek design. Minimalism at its finest.';
    popup.appendChild(message);

    overlay.appendChild(popup);
    document.body.appendChild(overlay);
  2. Open Your Authoring Tool: Launch your favorite authoring tool like Articulate Storyline, Adobe Captivate, or any other tool that supports JavaScript.
  3. Add a Button or Shape: - Insert a button and label it or insert a shape of your choice.
    - For a modern look, add a sleek info icon or a custom graphic to the shape or the button.
  4. Add a Trigger to the Button: - In Articulate Storyline:
    1. Select the button or shape.
    2. Open the Triggers Panel.
    3. Click “Create a New Trigger”.
    4. Set the options:
    - Action: Execute JavaScript
    - When: User clicks the button
    - In Adobe Captivate:
    1. Select the button or shape.
    2. Go to the Properties Panel.
    3. Under the Actions tab, set On Success to “Execute JavaScript”.
  5. Paste the JavaScript Code: In the JavaScript editor, paste the code you copied earlier.
  6. Modify the Pop-Up Content (Optional): Want to customize the heading and message? Here’s how:
    - To change the heading: Look for this line in the code: heading.textContent = 'Welcome!'; Replace 'Welcome!' with your own text (keep the double quotes!).

    - To change the message: Look for this line in the code: message.textContent = 'Experience a clean, sleek design. Minimalism at its finest.'; Replace the text inside the double quotes with your custom message.

    That’s it! Just be careful not to remove the double quotes or change the structure of the code.
  7. Preview or Publish and Test: - Preview Your Project:
    - In Storyline: Click “Preview” > “This Slide” or “Entire Project”.
    - In Captivate: Click “Preview” > “Project”.

    - Publish Your Project (Optional):
    - In Storyline: Go to Publish > Web or your preferred format.
    - In Captivate: Go to Publish > Publish to HTML5.

    - Test the Pop-Up:
    1. In the preview or published project, click the button or shape you added.
    2. The pop-up should appear with your custom heading and message.
    3. Ensure everything works as expected and the design looks sleek and polished.
