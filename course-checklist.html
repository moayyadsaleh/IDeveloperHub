Modern Course Checklist

This modern, interactive checklist is perfect for embedding directly into your e-learning courses. It’s a great way to keep your learners on track in self-paced courses! This tool lets students stay in control by tracking their progress through each module, task, or milestone.

DESCRIPTION

Instructional designers can easily embed the Interactive Checklist in their e-learning courses, such as Articulate Storyline, to help learners keep track of their progress in real time. The checklist updates dynamically as tasks are completed, providing a satisfying, interactive experience. With the smooth UI design, learners can check and uncheck items with ease, giving them full control of their learning journey.

The Interactive Checklist is designed to be flexible and easily customizable. It comes with a transparent background, so it seamlessly integrates with any course design. Place it on top of a shape or background element, and it will blend beautifully with the course's color scheme. One of the best features of this checklist is that it automatically saves progress. Once embedded, learners' data is saved across pages, ensuring that their progress is always up to date, no matter where they are in the course. This means no more worrying about lost progress when switching between tasks or returning after a break.

HOW TO USE

  1. Download the Zip File: Click the download button below to get the Interactive Checklist zip file. This file contains all the necessary code and assets for the checklist.
  2. Unzip the File: - On Windows: Right-click the downloaded .zip file and select "Extract All...". - On macOS: Double-click the .zip file to automatically unzip it. After unzipping, you'll get a folder named interactive-checklist.
  3. Open Your E-Learning Authoring Tool: Open your e-learning project in tools like Articulate Storyline, Adobe Captivate, or any other tool that supports web objects.
  4. Insert a Web Object: - In Articulate Storyline: - Go to the Insert tab and select Web Object. - In the Web Object dialog, click Browse. - Select the unzipped Interactive Checklist folder (make sure to select the whole folder, not individual files). - Click OK to insert the web object into your slide. - In Adobe Captivate: - Go to the Media menu and select Web Object. - Enter the path to the unzipped Interactive Checklist folder or browse to it. - Click OK to insert the web object.
  5. Position the Checklist: Move and resize the web object on your slide to fit your design. You can place it at the end of modules, tasks, lessons, or anywhere you want learners to track their progress.
  6. Progress is Automatically Saved: Don’t worry about losing data – the checklist automatically saves progress! In a self-paced course where you want learners to take charge of tracking their own progress, just create one checklist page, embed the tracker as shown above, and copy that same page wherever you need it in your course. Learners will be able to see and update their progress no matter where they are in the course.
  7. Preview and Test: - Preview your course to make sure the Interactive Checklist works correctly. - Learners will be able to check off tasks as they complete them.
