E-learning progress Tracker

I created this interactive tool for my fellow instructional designers to embed in e-learning courses. It’s perfect for self-paced learning to help learners take charge by manually tracking their progress after each module or task. It's inspired by the look and feel of Fitbit and uses fun visuals, sounds, and confetti to keep learners excited and show them how far they’ve come.

DESCRIPTION

I custom-coded ProgressTracker to help my fellow instructional designers create sleek, self-paced learning experiences. It’s easy to embed in tools like Articulate Storyline and works perfectly at the end of each module, task, or milestone. Learners can log progress with Complete, step back with Undo, or start fresh with Reset.

ProgressTracker has a transparent background so that it fits perfectly into any design. Place it over a shape or background element to match your course’s color scheme. Progress is saved automatically, so once you embed it on a progress page, you can copy and paste it anywhere in your course. Learners’ progress stays up-to-date across pages. No worries about losing data!

HOW TO USE

  1. Download the Zip File: Click the download button below to get the ProgressTracker zip file. This file contains all the necessary code and assets for the tracker.
  2. Unzip the File: - On Windows: Right-click the downloaded .zip file and select "Extract All...". - On macOS: Double-click the .zip file to automatically unzip it. After unzipping, you'll get a folder named progress-tracker.
  3. Open Your E-Learning Authoring Tool: Open your e-learning project in tools like Articulate Storyline, Adobe Captivate, or any other tool that supports web objects.
  4. Insert a Web Object: - In Articulate Storyline: - Go to the Insert tab and select Web Object. - In the Web Object dialog, click Browse. - Select the unzipped ProgressTracker folder (make sure to select the whole folder, not individual files). - Click OK to insert the web object into your slide. - In Adobe Captivate: - Go to the Media menu and select Web Object. - Enter the path to the unzipped ProgressTracker folder or browse to it. - Click OK to insert the web object.
  5. Position the Tracker: Move and resize the web object on your slide to fit your design. You can place it at the end of modules, tasks, lessons, or anywhere you want learners to track their progress.
  6. Progress is Automatically Saved: Don’t worry about losing data – the tracker automatically saves progress! In a self-paced course where you want learners to take charge of tracking their own progress, just create one progress page, embed the tracker as shown above, and copy that same page wherever you need it in your course. Learners will be able to see and update their progress no matter where they are in the course.
  7. Preview and Test: - Preview your course to make sure the ProgressTracker works correctly. - Learners will be able to hit Complete to log their progress, Undo to step back, and Reset to start over. - Verify that visual feedback, sound effects, and confetti animations trigger as expected.
  8. Track Up to 5 Items by Default: The tracker is set to track progress for up to 5 tasks, modules, or milestones by default. If you need to track more than 5 items, follow these steps to update the code: 1. Open the HTML File where your progress tracker is embedded using a text editor (such as Notepad, Visual Studio Code, or any code editor you prefer). 2. Navigate to the Script Section near the bottom of the file, between the ` ` tags. 3.Locate This Line in the Script: ```javascript const totalSections = 5; ``` 4. Update the Number: Change the number `5` to the number of tasks, modules, or milestones you want to track. - For example, to track 8 items, change it to: ```javascript const totalSections = 8; ``` 5.Save the FileAfter making the change, save the HTML file. 6. Test the Tracker: Open your e-learning course and check that the tracker correctly reflects the new number of items. Ensure that completing tasks updates the progress bar and that the visual feedback, sounds, and confetti animations work as expected. By updating this line, the tracker will now accommodate the new number of tasks, and progress will be saved automatically as learners interact with it.
