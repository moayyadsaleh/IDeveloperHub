Welcome to IDeveloperHUB

I created IDeveloperHub for my fellow instructional designers. It's a place where you can find ready-to-use code snippets that help you get things done faster and make your e-learning courses more engaging. If you’re working with Articulate Storyline, RISE, or Captivate, these tools are here to make your life easier. This space is built around the real challenges we run into as instructional designers. Just copy, paste, and you're good to go! And the best part? It’s completely FREE, and it always will be!

ABOUT THE PLATFORM

Each tool is carefully programmed with front-end and back-end technologies to to fit right into our e-learning courses. They’re designed to be reusable and customizable, which adds features that authoring tools don’t support natively, or would take too many layers and too much time to create. These code snippets tackle the real challenges we run into as instructional designers, so we spend less time troubleshooting and more time doing what we do best — creating meaningful, engaging learning experiences. Want to know more? Learn more about the platform.
